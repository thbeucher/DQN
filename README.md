# DQN


Implementation of:

  -DQN -> http://home.uchicago.edu/~arij/journalclub/papers/2015_Mnih_et_al.pdf
  
  -Double DQN -> http://arxiv.org/abs/1509.06461
  
  -Dueling network -> http://arxiv.org/abs/1511.06581
